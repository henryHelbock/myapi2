Books API - Summary Documentation
==================================

Overview
--------
A RESTful API built with FastAPI and Supabase (PostgreSQL) for managing a book collection.
The API supports full CRUD operations via the standard HTTP methods.

Live Server: https://myapi2-ps8m.onrender.com
Swagger UI:  https://myapi2-ps8m.onrender.com/docs
ReDoc:       https://myapi2-ps8m.onrender.com/redoc


Database Schema
---------------
Column           Type                    Description
id               bigint (auto)           Primary key
title            text (required)         Book title
author           text (required)         Author name
genre            text                    Genre
published_year   integer                 Year published
isbn             text                    ISBN number
inserted_at      timestamptz             Created timestamp
updated_at       timestamptz             Last updated timestamp


HTTP Methods
============

1. GET /books - List All Books
------------------------------
Returns all books ordered by ID.

Swagger UI: Open /docs, expand GET /books, click Try it out, then Execute.

curl:
  curl -X GET https://myapi2-ps8m.onrender.com/books

Postman: Set method to GET, enter URL, click Send.


2. GET /books/{book_id} - Get a Single Book
--------------------------------------------
Returns one book by its ID.

Swagger UI: Expand GET /books/{book_id}, enter an ID, click Execute.

curl:
  curl -X GET https://myapi2-ps8m.onrender.com/books/1

Postman: Set method to GET, enter URL with ID, click Send.


3. POST /books - Create a New Book
------------------------------------
Creates a new book. Requires title and author; other fields are optional.

Swagger UI: Expand POST /books, click Try it out, fill in the JSON body, click Execute.

curl:
  curl -X POST https://myapi2-ps8m.onrender.com/books \
    -H "Content-Type: application/json" \
    -d '{"title":"Brave New World","author":"Aldous Huxley","genre":"Dystopian","published_year":1932,"isbn":"978-0060850524"}'

Postman: Set method to POST, enter URL, set Body to raw JSON with the book data, click Send.


4. PUT /books/{book_id} - Replace a Book (Full Update)
-------------------------------------------------------
Replaces all fields of a book. Title and author are required.

Swagger UI: Expand PUT /books/{book_id}, enter the ID and full JSON body, click Execute.

curl:
  curl -X PUT https://myapi2-ps8m.onrender.com/books/1 \
    -H "Content-Type: application/json" \
    -d '{"title":"The Great Gatsby (Revised)","author":"F. Scott Fitzgerald","genre":"Classic Fiction","published_year":1925,"isbn":"978-0743273565"}'

Postman: Set method to PUT, enter URL with ID, set Body to raw JSON with all fields, click Send.


5. PATCH /books/{book_id} - Partial Update
--------------------------------------------
Updates only the fields you provide. At least one field must be included.

Swagger UI: Expand PATCH /books/{book_id}, enter the ID and partial JSON body, click Execute.

curl:
  curl -X PATCH https://myapi2-ps8m.onrender.com/books/1 \
    -H "Content-Type: application/json" \
    -d '{"genre":"American Classic"}'

Postman: Set method to PATCH, enter URL with ID, set Body to raw JSON with fields to update, click Send.


6. DELETE /books/{book_id} - Delete a Book
--------------------------------------------
Deletes a book by its ID.

Swagger UI: Expand DELETE /books/{book_id}, enter the ID, click Execute.

curl:
  curl -X DELETE https://myapi2-ps8m.onrender.com/books/1

Postman: Set method to DELETE, enter URL with ID, click Send.
